Questão 3 


a) 1, 3, 5, 7, ___

    A lógica aqui parece ser adicionar 2 a cada número anterior. Portanto, o próximo número seria 9.

b) 2, 4, 8, 16, 32, 64, ____

    Parece que cada número está sendo multiplicado por 2. Portanto, o próximo número seria 128.

c) 0, 1, 4, 9, 16, 25, 36, ____

    Essa sequência representa o quadrado dos números naturais de 0 a 6. O próximo número seria o quadrado de 7, que é 49.

d) 4, 16, 36, 64, ____

    Esses números são quadrados de números pares consecutivos. O próximo número seria o quadrado de 8, que é 64.

e) 1, 1, 2, 3, 5, 8, ____

    Esta é a sequência de Fibonacci, onde cada número é a soma dos dois anteriores. Portanto, o próximo número seria 13.

f) 2, 10, 12, 16, 17, 18, 19, ____

    Parece que os números estão aumentando de forma variada. Uma possível lógica é que cada número está sendo somado ao número anterior e subtraído de 1. Então, o próximo número seria 19 + 18 - 1 = 36.

Portanto, os próximos elementos seriam:
a) 9
b) 128
c) 49
d) 100
e) 13
f) 36
